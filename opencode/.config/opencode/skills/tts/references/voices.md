# Available Kokoro Voices

These voices are high-quality and available for the `tts` tool.

| ID | Gender | Description |
|---|---|---|
| `af_bella` | Female | Soft, clear, default. |
| `af_sarah` | Female | Professional, energetic. |
| `af_nicole` | Female | Calm, deliberate. |
| `af_sky` | Female | Youthful, bright. |
| `am_adam` | Male | Deep, authoritative. |
| `am_michael` | Male | Warm, friendly. |

**Usage:** `tts "text" voice_id`
